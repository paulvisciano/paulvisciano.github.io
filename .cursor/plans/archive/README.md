# Archived plans

Plans in this folder have been **executed** (shipped, completed, superseded, or cancelled). They are kept for reference and history.

**Process:** See [PLAN-ARCHIVING-STRATEGY.md](../PLAN-ARCHIVING-STRATEGY.md) for when and how to archive plans.

**Layout:** Files can live directly here (flat) or in date subfolders like `2026/02/` if you add them later.

**Re-open:** Move a file back to `../` and update its frontmatter if you want to work from it again.
