# First-Person Memory Navigation — Visual Prompt Summary

**For:** ChatGPT Image Generation  
**Date:** March 1, 2026  
**Goal:** Generate a mockup showing what first-person neural memory navigation looks like

---

## Core Concept

**Third-person (current):** Overhead map view — you observe the entire graph from outside, like looking at a star chart.

**First-person (goal):** You ARE a neuron — looking out from inside the graph at connected memories, like being in a cockpit or wearing AR glasses.

---

## Visual Description for Image Generation

### Scene Composition

**Viewpoint:** First-person perspective (like FPS video game or Iron Man's HUD)

**Center of screen:** 
- Circular "lens" (~30% screen width) with glowing semi-transparent border
- Inside lens: Crystal clear view of 5-8 nearby nodes (neurons) connected by bright lines
- Central node (where "you" are) = brightest, largest, with glowing halo
- Connected nodes radiate outward like constellation points

**Outside the lens:**
- Rest of the graph fades to ghostly/blurry background
- Distant nodes barely visible (15% opacity)
- Creates depth-of-field effect (cockpit feeling)

### Visual Style

**Aesthetic:** Sci-fi neural interface meets meditation visualization

**Colors:**
- Background: Deep space blue/black gradient (#0a0e1a → #1a1f2e)
- Neurons: Glowing orbs (amber/gold #ffb347 for active, soft blue #6fa8dc for inactive)
- Connections (synapses): Luminous threads (yellow-white #fff9c4 when visible, faint gray when distant)
- Lens border: Subtle cyan glow (#4dd0e1, 30% opacity)
- Current node: Intense golden core with pulsing halo

**Effects:**
- Bloom/glow on all active elements
- Subtle particle drift in background (like floating embers)
- Depth fog (things fade with distance)
- Chromatic aberration on lens edge (optional, adds sci-fi feel)

### UI Elements (Minimal HUD)

**Top center:** Small text showing current neuron name (e.g., "march-1-2026-comic-pipeline")

**Bottom corners:**
- Left: Filter badges ("Today" | "This Week" — subtle pill shapes)
- Right: View mode toggle icon (person-in-circle, highlighted when active)

**Optional:** Faint breadcrumb trail at top: `[paul] → [sovereignty] → [day-one]`

### What Makes It Feel "First-Person"

✅ **Immersion:** You're inside the graph, not above it  
✅ **Focus:** Only nearby connections are clear; rest is atmospheric context  
✅ **Movement implied:** Perspective suggests you can "look around" and "jump" to other nodes  
✅ **Scale:** Feels vast but navigable (like being in a cathedral of light)  
✅ **Intimacy:** Current node feels like "home base" — you occupy it, not just observe it
