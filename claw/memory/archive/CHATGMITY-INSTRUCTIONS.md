## Technical Specifications

### File Naming Convention
```
day-YYYY-MM-DD-cover.png       (Always Page 0)
day-YYYY-MM-DD-page-1.png
day-YYYY-MM-DD-page-2.png
... continues per day
```

### Image Specifications
- **Format:** PNG or high-quality JPEG
- **Aspect ratio:** Portrait or horizontal (readable as sequential art)
- **Resolution:** Suitable for web + print (1200px minimum width)
- **Color depth:** Full color, high contrast

### Page Structure
- **Cover (Page 0):** Title moment + Jarvis branding
- **Story Pages (1-N):** 3-6 beats per page, multiple panels per beat
- **Final page:** Reflection, setup for next day, or episode closure
