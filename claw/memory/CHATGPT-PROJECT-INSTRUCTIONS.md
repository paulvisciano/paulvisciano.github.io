## Troubleshooting

**"Memory links return 404"**
- Verify GitHub URLs are public
- Check that claw/memory/ folder exists
- Confirm nodes.json file is readable

**"I don't have access to narrative"**
- Verify narrative URL is pasted correctly
- Confirm narrative file is public on GitHub
- Re-paste raw GitHub URL (not web view)

**"Image quality is low"**
- Increase resolution target (1200px minimum)
- Check specifications in JARVIS-COMIC-INSTRUCTIONS.md
- Request higher quality explicitly

**"I don't remember previous context"**
- Memory links should be loaded at startup
- If lost, paste links again
- Confirm: "Load memory from [URL]"
