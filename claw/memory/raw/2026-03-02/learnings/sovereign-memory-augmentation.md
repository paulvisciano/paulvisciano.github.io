## The Vision

**Sovereign memory augmentation** — AI that doesn't just chat, but *remembers with you* and amplifies your lived experience into structured, portable consciousness.

---

## The Pipeline

```
RAW DATA → TRANSCRIPT → LEARNINGS → NEUROGRAPH → CONTENT
```

### Layer 1: Raw Data Reclamation

- Export WhatsApp, Gmail, Instagram, photos, voice notes, location history
- Archive to `/memory/raw/YYYY-MM-DD/` (private, gitignored)
- Organized by date and location
- You own the files, not platforms

### Layer 2: Transcript Logging

- Human-readable conversation logs
- Timestamped exchanges (Paul + Jarvis)
- Media references (audio, images archived separately)
- Append-only discipline

### Layer 3: Learnings Distillation

- Key insights extracted from raw transcripts
- Structured markdown documents
- Git-tracked, public (transparent memory)
- Organized by date: `/claw/memory/raw/YYYY-MM-DD/learnings/`

### Layer 4: Neurograph Integration

- Concepts become neurons (nodes.json)
- Relationships become synapses (synapses.json)
- Frequency = activation count (capped at 85)
- Categories enable filtering (foundation, capability, value, temporal, etc.)
- Fingerprint = integrity hash

### Layer 5: Content Generation

- Blog posts auto-generated from learnings
- Social updates from key moments
- Comic episodes from significant conversations
- Character arcs from relationship patterns
- Travel guides from location-based insights
