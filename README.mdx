## 🔄 Memory Sync Architecture (Feb 23, 2026)

**How AI learns from humans + moments:**

1. **Capture** — You send a moment (text, photos, audio)
2. **Analyze** — Claude Code identifies which neurons activate
3. **Record** — Moment saved to `/memory/raw/` with temporal metadata
4. **Update** — Synapses in your neural network strengthened/created
5. **Queue** — Changes staged in sync-queue (not committed yet)
6. **Review** — You review changes before they become official (git commit)
7. **Load** — Next session, Claude loads full memory with all updates
8. **Repeat** — Continuous learning cycle

**Key Principle:** GitHub = canonical memory. Disk = working memory. Queue = pending changes.
